# Claude PR Review

이 프로젝트는 GitHub Actions를 활용하여 Claude Code를 통한 자동화된 PR 리뷰와 코멘트 해결을 제공합니다.

## 워크플로우 액션

### 1. PR Comment Resolver (`pr-comment-resolver.yml`)

PR 코멘트에서 `@claude`로 시작하는 요청을 자동으로 처리하는 워크플로우입니다.

**기능:**

- PR 코멘트나 리뷰 코멘트에서 `@claude` 명령어 감지
- Claude Code를 사용하여 요청된 작업 수행
- 파일 수정이 필요한 경우 자동으로 커밋하고 푸시
- 처리 결과를 PR에 코멘트로 응답

**트리거:**

- 이슈 코멘트 생성 시 (`@claude`로 시작하는 경우)
- PR 리뷰 코멘트 생성 시 (`@claude`로 시작하는 경우)

### 2. PR Reviewer (`pr-reviewer.yml`)

PR에 대한 자동화된 코드 리뷰를 수행하는 워크플로우입니다.

**기능:**

- 코드 품질, 보안, 성능, 모범 사례 등에 대한 종합 리뷰
- 수정된 파일만 집중하여 리뷰 수행
- 리뷰 결과를 구조화된 JSON 형식으로 생성
- PR에 리뷰 코멘트와 승인/변경 요청 이벤트 제출

**트리거:**

- PR 생성 시 (제목이 `.`로 시작하지 않는 경우)
- PR 코멘트에서 `/review` 명령어 입력 시
- 수동 실행 (workflow_dispatch)

## 설정

1. `ANTHROPIC_API_KEY` 시크릿을 GitHub 리포지토리에 설정하세요.
2. 필요한 권한들을 워크플로우에 부여하세요.

## 사용법

### PR 코멘트 해결

PR 코멘트에 다음과 같이 작성하세요:

```
@claude [요청사항]
```

### PR 리뷰 요청

PR 코멘트에 `/review`를 입력하거나, PR을 생성하여 자동으로 리뷰를 받으세요.
